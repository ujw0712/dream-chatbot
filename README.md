## 드림이에게 물어보세요!

타 팀에서 개발팀에게 질문/요청 시 신속한 처리를 위한 챗봇입니다.  
요청자가 원하는 정보를 찾아 알려주고(Notion), 기타 질문을 등록하면 담당자(참조: 개발팀)에게 메일을 보내주는 시스템입니다.
